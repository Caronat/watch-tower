# Watch Tower (crypto App)

Application permettant de suivre l'évolution des cryptomonnaies.

Les données proviennent de l'[API du site CoinGecko](https://www.coingecko.com/fr/api)
Pour les graph, j'ai utilisé la librairie [<Recharts />](https://recharts.org/en-US)

Pour tester l'application, c'est par [ici](https://watch-tower.antonycaron.fr)
